== Name

bite-bugzilla-attachment-update - update attachment metadata

== Synopsis

*bite bugzilla attachment update* [<options>] [id[,...]]::
    Update attachment metadata.

*bite bugzilla attachment update* [-h|--help]::
    Output help information.

== Description

Update attachment metadata.

include::include/global-options.adoc[]

== Attachment options

*-c, --comment* <value>::
    Comment related to the attachment.

*-d, --description* <value>::
    A short description of the attachment.

*-f, --flags* <value>[,...]::
    Modify attachment flags.

*-m, --mime* <type>::
    MIME type of the attachment.

*-n, --name* <value>::
    File name of the attachment.

*-o, --obsolete* [<bool>]::
    Toggle obsolete status.

*-p, --patch* [<bool>]::
    Toggle patch status.

*-P, --private* [<bool>]::
    Toggle private status.

== Arguments

<id[,...]>::
    Attachment IDs.
+
Taken from standard input when `-`.
