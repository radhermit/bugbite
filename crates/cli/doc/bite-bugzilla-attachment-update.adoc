== Name

bite-bugzilla-attachment-update - update attachment metadata

== Synopsis

*bite bugzilla attachment update* [<options>] [id[,...]]::
    Update attachment metadata.

*bite bugzilla attachment update* [-h|--help]::
    Output help information.

== Description

Update attachment metadata.

include::include/global-options.adoc[]

== Attachment options

*-c, --comment* <value>::
    Update comment.

*-d, --description* <value>::
    Update description.

*-f, --flags* <value>[,...]::
    Update flags.

*-m, --mime* <type>::
    Update MIME type.

*-n, --name* <value>::
    Update file name.

*-o, --obsolete* [<bool>]::
    Update obsolete status.

*-p, --patch* [<bool>]::
    Update patch status.

*-P, --private* [<bool>]::
    Update private status.

== Arguments

<id[,...]>::
    Attachment IDs.
+
Taken from standard input when `-`.
