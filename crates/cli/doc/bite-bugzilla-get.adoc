== Name

bite-bugzilla-get - get bugs

== Synopsis

*bite bugzilla <get|g>* [<options>] <ids>::
    Get bugs.

*bite bugzilla <get|g>* [-h|--help]::
    Output help information.

== Description

Get bugs.

include::include/global-options.adoc[]

== Get options

*-A, --no-attachments*::
    Disable attachments.

*-C, --no-comments*::
    Disable comments.

*-H, --no-history*::
    Disable history.

*-b, --browser*::
    Open bugs in a browser.
+
This functionality uses `xdg-open` to open URLs.

== Arguments

<ids>::
    IDs or aliases of bugs to fetch.
+
Taken from standard input when `-`.

== Examples

.Get gentoo bug 123:
 bite -c gentoo get 123

.Get gentoo bug 123 and only output field values:
 bite -c gentoo get 123 -ACH

.Get all gentoo bugs created in the last day:
 bite -c gentoo search -c 1d -f id | bite -c gentoo get -
