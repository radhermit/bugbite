== Name

bite-bugzilla-history - get bug changes

== Synopsis

*bite bugzilla history* [<options>] <ids>::
    Fetch and output bug changes.

*bite bugzilla history* [-h|--help]::
    Output help information.

== Description

Get bug changes.

include::{docdir}/common/global-options.adoc[]

== History options

*-c, --created*::
    Restrict by creation time.
+
.Changes to bug #123 in the last day:
 bite bugzilla history -c 1d 123

*-R, --creator*::
    Restrict by creator.
+
.Changes to bug #123 by user1:
 bite bugzilla history --creator user1 123

== Arguments

*<ids>*::
    Bug IDs or aliases of changes to fetch.
+
Taken from standard input when `-`.
